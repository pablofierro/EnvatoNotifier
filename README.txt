Minimalist Envato sales notifier written in objective-c for OS X.

This project was created with learning purposes only, you can do whatever you want with it.

It shall be a good point of start for those who are starting to learn objective-c and cocoa.

The following was implemented in the project:

1-) HTTP Data fetching
2-) JSON Parsing - using https://github.com/gabriel/yajl-objc
3-) GROWL notifications using the growl developer framework.

This app is only functional for Envato marketplaces users that wish to monitor their sales activity using MAC OS X.
